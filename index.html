imapcp by gtozzi

Copy emails and folders from an IMAP account to another

Copy emails and folders from a source IMAP account to a target IMAP account - Creates missing folders and skips existing messages (using message-id) - Source IMAP is always accessed READ-ONLY - Allowes exclusion of folders based on filter

Install

You'll need Python 2.7 or greater. just download lastest imapcp.py and run it from the command line with --help switch.
